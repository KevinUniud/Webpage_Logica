Equivalenze
P -> Q ↔ ¬P ∨ Q
P -> Q ↔ ¬Q -> ¬P
P ∨ Q ↔ ¬P -> Q
P ∧ Q ↔ ¬(P -> ¬Q)
¬(P -> Q) ↔ P ∧ ¬Q
¬∀x P ↔ ∃x ¬P
¬∃x P ↔ ∀x ¬P
Ordine di precedenza
  1. 1. ( )   parentesi
  2. 2. ¬   negazione
  3. 3. ∧   congiunzione (E)
  4. 4. ∨   disgiunzione (O)
  5. 5. →   implicazione
ⓘ Gli operatori con precedenza più alta vengono valutati per primi.
Esempio: ¬P ∧ Q → R equivale a ((¬P) ∧ Q) → R.
